# Relationships

```
        栞（記憶／オレンジ）
          ▲
          │ 思い出せなくなることへの恐れ
          │
         陽（現実／青）
          │
          │ 契約 ── 見送り
          ▼
        クラ（感情銀行／ラブドガンの所持者）

        小面（観測者／黒）
          ‥ 陽・クラ双方を無言で見ている（介入しない）
```

## 陽 ⇄ 栞
現在の関係ではなく、記憶としての関係。
陽が失いたくないもの、そのものであり、物語の感情的な中心。
画面上で直接的な対話が長く続くことはない（`Shiori.md` 参照）。

## 陽 ⇄ クラ
契約者と、それを執行する者。
クラは陽を裁かず、急かさず、ただ提示し、最終的に見送る。
上下関係ではなく、機能としての非対称性。

## 陽 ⇄ 小面
直接的な関わりはほとんどない。
小面は陽の選択を見ているだけで、介入しない。
物語の終盤、陽が制度の外で自分の意思により受容に至ったことを、
小面の存在（無言のまま画面に映ること）で示す。

## クラ ⇄ 小面
共に感情銀行という制度に属する（あるいは隣接する）が、役割は異なる。
クラは執行者、小面は観測者。両者の間に対話は発生しない。

## 三色との対応
- 陽＝青（現実）
- 栞＝オレンジ（記憶）
- クラ＝白（感情世界／感情銀行に属する）
- 小面＝黒（唯一許された色）
