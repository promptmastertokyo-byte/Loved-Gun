# LOVED GUN — Model Sheets

このディレクトリは、制作で使用するキャラクター／プロップの正式モデルシートを管理する。

## Directory Structure

```text
characters/model-sheets/
├── yo/
├── kura/
├── shiori/
├── koomote/
├── loved-gun/
└── README.md
```

## File Naming

```text
<subject>_<type>_v<version>.<ext>
```

Examples:

- `yo_turnaround_v1.png`
- `yo_expression_v1.png`
- `yo_emotional-collapse_v1.png`
- `kura_turnaround_v1.png`
- `shiori_expression_v1.png`
- `koomote_mask-detail_v1.png`
- `loved-gun_prop-sheet_v1.png`

## Required Sheets

### Characters

- Turnaround: front / 3/4 / profile / back
- Expression sheet
- Costume detail
- Hand and acting reference when needed

### Special Production Sheets

- Yo emotional-collapse acting sheet
- Shadow-delay behavior sheet
- Koomote mask and crack detail
- Loved Gun prop turnaround / materials / activation detail

## Status Rule

画像を追加したら `characters/CHARACTER_REFERENCE_TRACKER.md` も更新する。

- `design status`
- `reference version`
- `file path`
- `approval status`
- `last updated`

レイアウトでは承認済みモデルシートのみを使用する。
