## プロット（500字程度）

灰色の空に細い雨が降る朝。高校生の陽は、同級生だった栞を亡くしてから、色褪せた一枚の写真を胸ポケットにしまったまま歩いている。交差点で立ち止まった陽は、自分の足元の影がわずかに遅れて動くことに気づく。導かれるように迷い込んだ路地の奥、古い看板を掲げた店「感情銀行」。店主のクラは、壁のガラスケースに眠る古い拳銃「ラブドガン」を差し出す。人を撃つための銃ではない、見ないようにしているものを撃ち抜くための銃だと。陽が白い壁に向けて放った一発は、壁ではなく陽自身の足元から人の形をした「影」を引きずり出す。栞だと思ったそれは、陽自身の感情そのものだとクラは告げる。逃れようとする陽に、影は写真へ手を伸ばし、部屋を黒い粒子が這う。屋上で交わした「また来ようね」の記憶と、「忘れてもいいよ」と笑った栞の言葉を思い出しながら、陽は最後の一発を、他の誰でもない自分の胸へ向ける。「これは痛いよ」とクラが言う。陽は答える。「忘れない。君と一緒に生きる」。悲しみは消えない。それでも陽は影を連れたまま、雨上がりの街へ一歩を踏み出す。

（約460字）

## ログライン

亡くした恋人を「忘れるための銃」を渡された少年は、その銃口を自分自身の悲しみへ向け、痛みを抱えたまま生きることを選ぶ。

---

## イメージした楽曲

台詞・環境音（雨、電車、信号機、鈴、ガラスの砕ける音）を主体とし、音楽は要所にのみ最小限で挿入する方針（世界観バイブルの音響設計方針を踏襲）。

- **参考アーティスト／曲調**
  - Ólafur Arnalds — ミニマルなピアノ＋ストリングス。処置室シーンの静けさの基調
  - ヨルシカ「花に亡霊」「だから僕は音楽を辞めた」的な、儚さと痛みが同居する歌なしのアレンジ
  - World's End Girlfriend — 環境音とクラシック的な旋律の融合。影が現れる瞬間の不穏さ
  - 坂本龍一の晩年のソロピアノ曲のような、余白の多い旋律。ラストの「歩き出す」場面のみ短いピアノモチーフ
- **使用方針**：本編中盤（処置室での対峙〜自分を撃つ場面）まで音楽はほぼ排し、ラストカットの一歩だけに短いピアノモチーフを添える。

## 想定所在地

作品内で地名は明示しない架空の日本の地方都市（現代）という設定を維持しつつ、美術・ロケハンの参照先としては以下を想定。

- **交差点・商店街のシーン**：谷根千（谷中・根津・千駄木）や都電荒川線沿線のような、昭和の面影が残る下町の商店街・路地
- **感情銀行の外観**：古い建具店・骨董店のような、間口の狭い木造家屋（上記エリアや尾道・函館の坂道商店街も参照候補）
- **学校・屋上（回想）**：郊外の公立高校のような、簡素で開放感のある屋上（フェンス・給水塔）
- 実写ロケではなく背景美術の資料撮影・参考写真収集を想定した候補地

## 予算見積り（短編シナリオ版）

既存の`budget/BUDGET_10M_ASSUMPTIONS.md`は旧版（20分・ノウメン登場・180〜230カット）を前提とした1,000万円見積り。
本シナリオは登場人物3〜4名（陽・栞・クラ・影）、主要ロケーション実質3箇所（交差点／感情銀行内外／屋上回想）、
上映時間目安6〜8分、カット数目安70〜100カットと大幅に小規模なため、別枠で見積る。

| 項目 | 概算 |
|---|---:|
| 脚本・監督準備 | 200,000円 |
| キャラクター・プロップ設定（影の動き含む） | 300,000円 |
| 絵コンテ・レイアウト | 400,000円 |
| 原画・作画監督 | 1,200,000円 |
| 動画・仕上げ | 600,000円 |
| 背景美術 | 500,000円 |
| 撮影・コンポジット（影のパーティクルVFX、雨・反射表現） | 700,000円 |
| 音響・声優（3〜4役）・音楽 | 400,000円 |
| 編集・字幕・納品・予備費 | 300,000円 |
| **合計目安** | **4,600,000円** |

### 想定と前提
- 影のパーティクル演出・ガラス破裂・鏡面反射（濡れた路面）が本作最大のVFXコストであり、撮影・コンポジット費を厚めに確保
- 群衆カットは最小限（交差点シーンのみ）、背景美術は3系統（商店街／感情銀行内装／屋上）で使い回し前提
- 声優は陽・栞・クラの3役を基本とし、影は台詞なし（VFX＋芝居のみ）と想定
- 10〜15%の予備費を最終納品まで確保する方針は旧見積りと同様
