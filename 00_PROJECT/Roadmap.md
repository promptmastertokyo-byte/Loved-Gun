# Roadmap

『Loved Gun』完成までの大枠フェーズ。
日付は `Schedule.md` で管理し、ここではフェーズと成果物のみを定義する。

## Phase 0 — Development（企画開発）
- [x] Vision / Director Statement 確定
- [x] World Bible（感情銀行 / ラブドガン / ノウメン）確定
- [ ] Character Bible 確定（陽・クラ・小面・栞・関係性）
- [ ] Theme / Visual Bible 確定
- [ ] Production Secrets（内部設定資料）確定

## Phase 1 — Writing（脚本）
- [ ] Beat Sheet 確定
- [ ] Treatment（プロット）確定
- [ ] Screenplay 初稿
- [ ] Screenplay 改稿（決定稿）
- [ ] Dialogue 抜粋・台詞リスト確定

## Phase 2 — Pre-Production（絵コンテ / 美術 / 音）
- [ ] 04_STORYBOARD：絵コンテ作成
- [ ] 05_ART：美術設定（3色設計の色見本、背景美術、ノウメンのデザイン）
- [ ] 06_SOUND：環境音リスト収録・選定（雨/信号/電車/風/足音/銀行のベル）
- [ ] キャスティング（陽・クラ・小面・栞の声）

## Phase 3 — Financing（資金）
- [ ] 07_BUDGET：予算策定
- [ ] 助成金・映画祭ファンド候補リストアップ

## Phase 4 — Production（制作）
- [ ] 作画・撮影
- [ ] 効果音・環境音の実収録／選定音源の仮当て
- [ ] 仮編集（オフライン編集）

## Phase 5 — Post-Production（仕上げ）
- [ ] 本編集
- [ ] MA（整音・最終ミックス）

- [ ] 完パケ

## Phase 6 — Festival（映画祭出品）
- [ ] 09_FESTIVAL：出品先リストアップ・締切管理
