# LOVED GUN LP — 運用メモ

更新日：2026-07-13

## 1. 対象

このディレクトリは、自主制作2D短編アニメーション映画『LOVED GUN / ラブドガン』の公式コンセプトLPです。

- 配置：`lp/`
- 技術：Next.js 16 / React 19 / TypeScript / Tailwind CSS
- 対象作品：上映時間9分の短編シナリオ版
- 対象外：長編・シリーズ寄り版

## 2. 確定設定

### ログライン

> 亡くした恋人を「忘れるための銃」を渡された少年は、その銃口を自分自身の悲しみへ向け、痛みを抱えたまま生きることを選ぶ。

### キャッチコピー

> 忘れたいのは、君じゃなかった。

### 登場人物

- **陽（ひなた・17）**：高校生。亡くした栞の写真を持ち歩いている
- **栞（しおり・17）**：陽の同級生。故人。回想の中にだけ現れる
- **クラ（29）**：「感情銀行」の主。ラブドガンを扱う
- **影**：陽自身の感情の具現。顔はなく、陽と同じ動きを半拍遅れて追う

### 作品情報

- 形式：2D短編アニメーション映画
- 上映時間：9分
- 舞台：現代日本のどこかにある架空の地方都市／都市部の路地裏
- テーマ：喪失を受け入れる勇気／忘れることと前へ進むことの違い／悲しみを抱えたまま歩くこと
- 地名、「感情銀行」、ラブドガン、登場人物、出来事はすべて創作

## 3. 重要な禁止事項

- **小面／ノウメンは短編版に登場しない。** コピー、FAQ、人物紹介、画像説明、メタ情報へ追加しない
- 旧設定の「春川陽（はるかわ・よう）」を使わない。短編版では「陽（ひなた）」とする
- 旧上映時間の「6〜8分」「20分」を使わない
- 12話シリーズやパイロット作品として説明しない
- 栞を生存人物として説明しない。栞は故人で、回想の中にだけ現れる

## 4. 確定プロット

灰色の空に細い雨が降る朝。高校生の陽は、同級生だった栞を亡くしてから、色褪せた一枚の写真を胸ポケットにしまったまま歩いている。交差点で立ち止まった陽は、自分の足元の影がわずかに遅れて動くことに気づく。導かれるように迷い込んだ路地の奥、古い看板を掲げた店「感情銀行」。店主のクラは、壁のガラスケースに眠る古い拳銃「ラブドガン」を差し出す。人を撃つための銃ではない、見ないようにしているものを撃ち抜くための銃だと。陽が白い壁に向けて放った一発は、壁ではなく陽自身の足元から人の形をした「影」を引きずり出す。栞だと思ったそれは、陽自身の感情そのものだとクラは告げる。逃れようとする陽に、影は写真へ手を伸ばし、部屋を黒い粒子が這う。屋上で交わした「また来ようね」の記憶と、「忘れてもいいよ」と笑った栞の言葉を思い出しながら、陽は最後の一発を、他の誰でもない自分の胸へ向ける。「これは痛いよ」とクラが言う。陽は答える。「忘れない。君と一緒に生きる」。悲しみは消えない。それでも陽は影を連れたまま、雨上がりの街へ一歩を踏み出す。

## 7. デザイン方針

- 映画公式サイトらしい静かで内省的なトーン
- 青空、雨に濡れた青、白、灰、ミントを中心に使う
- 明朝体の和文見出しと、簡潔なゴシック体本文を組み合わせる
- 説明過多にせず、余白、光、反射、線画を活かす
- PCとスマートフォンの両方で横スクロールを発生させない
- `prefers-reduced-motion` を尊重する

## 8. 開発と検証

```bash
cd lp
npm ci
npm run dev
npm run lint
npm run build
npm audit --omit=dev
```

公開前に次を確認すること。

- `npm run lint` と `npm run build` が成功する
- `npm audit --omit=dev` が0件である
- PC／スマートフォンで表示崩れや横スクロールがない
- PVとFAQが操作できる
- 「小面」「6〜8分」「20分」「全12話」「春川陽」が残っていない
- 上映時間がすべて9分で統一されている
